## 4. 역할 분담

### 😎 문재경

- **실시간 채팅**
  - WebSocket 과 Redis Pub/Sub 을 이용한 실시간 채팅 기능 구현
  - 채팅, 채팅방 관련 기능 구현
  - MongoDB 를 이용한 채팅 기록 저장
- **문의**
  - 문의 관련 API 구현
- **멤버**
  - 멤버 관련 API 구현
  - Security 를 통한 유저 권한 체크 및 인증 구현
- **설정**
  -  Swagger 설정
  -  공통 응답 메서드 구현

<br>

### 🙃 최지우

- **게시글**
  - 게시글 CRUD 구현
  - 타인 게시글 조회 구현

- **찜**
  - 찜 목록에 게시글 추가 기능 구현
